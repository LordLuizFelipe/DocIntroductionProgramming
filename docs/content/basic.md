### **COMANDOS BÁSICOS**

#### **PRINT**

>Função print()
``` python
print()
```
>>$

# 

>Função print com um argumento
``` python
print(’Meu nome é arretado!’)
```
>>Meu nome é arretado!

# 

#### **INPUT**

>Função input()
``` python
nome = input("Digite o seu nome: ")
```
>>Digite o seu nome: 

<p>
A função input recebe como argumento um valor em string que será mostrado ao usuário, e também retorna um valor string.
</p>
